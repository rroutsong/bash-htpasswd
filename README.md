
# bash-htpasswd

bash executable for htpasswd generation

### Prerequisites

openssl

### Installing

in a bash terminal:
>git clone https://github.com/rroutsong/bash-htpasswd.git
>cd bash-htpasswd
>chmod +x htpasswd
>./htpasswd [username] [password] [file location]

or just wrap in a function and add to your .bashrc
